Before You Start

Please read these instructions carefully before entering the virtual debate room.

  • Use a laptop or desktop computer for the best experience.
  • Open this website using Google Chrome.
  • Allow microphone access when the browser asks for permission.
  • Use a headset or earphones if available.
  • Complete the units in order: Unit 1 → Unit 2 → Unit 3 → Unit 4.
  • Record your answer, listen to your recording, and try again if needed.
  • Speak naturally. Do not read a full script.
