Virtual Debate Room for English Speaking Practice

This platform guides students through two complete debate motions. Each motion consists of four speaking turns: Affirmative arguments, Negative rebuttal and arguments, Affirmative rebuttal and arguments, and Negative final rebuttal.

Start Motion 1 Start Motion 2
Student Login Students enter name, ID, and class before recording their speech.
3D Debate Room Each unit uses a virtual stage with 3D avatars for debate practice.
Recording + Transcript Students can record their voice and edit the speech transcript.
Feedback + ZIP Students generate feedback and download a submission folder with their name.

How Students Use Each Unit

Each unit follows the same workflow, so students can focus on speaking and argument development.

1. Login Enter name, student ID, and class. Save the student login before recording.
2. Prepare Read the motion and role instruction, then write short notes in the preparation boxes.
3. Record Click Start Recording, speak clearly, then click Stop Recording.
4. Feedback Edit the transcript if needed, then click Generate Feedback.
5. Download Download the ZIP submission folder and submit it to the lecturer.
